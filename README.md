# Cak Yusop Cafe Management System

This is a premium Cafe Management System with POS, Inventory, and Analytics functionalities.

## Technology Stack
- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **API**: Node.js Backend (Internal/External)

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build and Deploy

To create a production-ready static build:

```bash
npm run build
```

The output will be in the `dist/` directory.

### Deploy on Vercel
This project is configured for a static Vite deployment on Vercel. Ensure the **Framework Preset** in your Vercel project settings is set to **Vite** or **Other**.
